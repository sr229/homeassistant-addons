# Logo/Icon Information

This add-on should have a `logo.png` file for better visual appearance in the Home Assistant add-on store.

## Requirements

- File name: `logo.png`
- Recommended size: 128x128 pixels or larger
- Format: PNG with transparency preferred
- Content: Minecraft-themed image (e.g., grass block, creeper face, pickaxe)

## How to Add

1. Create or download a Minecraft-themed icon
2. Save it as `logo.png` in this directory
3. The icon will automatically appear in the Home Assistant add-on store

## Resources

You can find suitable icons at:
- [Minecraft Wiki](https://minecraft.fandom.com/wiki/Minecraft_Wiki)
- Create your own using pixel art tools
- Use a grass block or other recognizable Minecraft symbol
